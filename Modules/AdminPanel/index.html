Группы (JSON)

Ключи из current.conf, сгруппированные по секциям. Правки сохраняются в init.conf. Удалить ключ: используйте null или редактируйте весь файл.

Визуальные группы

Настройки из init.conf в виде переключателей, чисел и обычных полей. Сложные значения остаются JSON-полями, а сохранение меняет нужную секцию в init.conf.

Переключатели вместо true/false Включение источников, прокси и режимов меняется тумблером.
Поля вместо структурного JSON Порт, адреса, токены и таймауты редактируются отдельными строками.
JSON остаётся рядом Вкладки «Группы» и «init.conf» остаются технической базой и точкой контроля.
